Политика конфиденциальности

Защита персональных данных пользователей

1. Общие положения

Настоящая Политика конфиденциальности составлена в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

Оператор ставит своей важнейшей целью соблюдение прав и свобод человека при обработке его персональных данных, включая защиту прав на неприкосновенность частной жизни, личную и семейную тайну.

2. Основные понятия

Обработка персональных данных
Любые действия с данными: сбор, запись, систематизация, накопление, хранение, уточнение, извлечение, использование, передача, обезличивание, блокирование, удаление и уничтожение.
Оператор
SV EXPRESS — организация, самостоятельно организующая и осуществляющая обработку персональных данных.
Пользователь

Блокирование персональных данных
Временное прекращение обработки персональных данных (за исключением случаев, когда обработка необходима для уточнения данных).
Обезличивание персональных данных
Действия, в результате которых невозможно определить принадлежность персональных данных конкретному субъекту без использования дополнительной информации.

3. Права и обязанности Оператора

Оператор имеет право:

  • Получать от субъекта персональных данных достоверную информацию и/или документы, содержащие персональные данные
  • Продолжить обработку персональных данных при наличии законных оснований после отзыва согласия субъекта
  • Самостоятельно определять состав и перечень мер, необходимых для обеспечения выполнения обязанностей по защите персональных данных

Оператор обязан:

  • Предоставлять субъекту персональных данных по его просьбе информацию, касающуюся обработки его персональных данных
  • Организовывать обработку персональных данных в соответствии с законодательством Российской Федерации
  • Принимать необходимые правовые, организационные и технические меры для защиты персональных данных
  • Публиковать или иным образом обеспечивать неограниченный доступ к документу, определяющему политику в отношении обработки персональных данных

4. Права и обязанности субъектов персональных данных

Субъекты персональных данных имеют право:

  • Получать информацию, касающуюся обработки своих персональных данных
  • Требовать от оператора уточнения персональных данных, их блокирования или уничтожения
  • Отозвать согласие на обработку персональных данных
  • Обжаловать неправомерные действия или бездействие оператора

Субъекты персональных данных обязаны:

  • Предоставлять достоверные сведения о себе
  • Своевременно сообщать об изменении своих персональных данных

5. Принципы обработки персональных данных

  • Обработка персональных данных осуществляется на законной и справедливой основе
  • Обработка ограничивается достижением конкретных, заранее определённых и законных целей
  • Не допускается объединение баз данных, содержащих персональные данные, обработка которых осуществляется в несовместимых между собой целях
  • Обеспечивается точность, достаточность и актуальность персональных данных
  • Хранение персональных данных осуществляется в форме, позволяющей определить субъекта, не дольше, чем требуют цели обработки

6. Цели обработки персональных данных

Обрабатываемые данные:

Фамилия, имя, отчество Электронный адрес Номера телефонов Адрес Дата рождения

Цель обработки:

Предоставление доступа к сервисам и материалам на сайте, оформление и исполнение заказов на доставку посылок.

Виды обработки:

Сбор, запись, систематизация, накопление, хранение, уточнение, извлечение, использование, передача, обезличивание, блокирование, удаление и уничтожение персональных данных.

7. Условия обработки персональных данных

Обработка персональных данных осуществляется при наличии хотя бы одного из следующих условий:

  • Согласие субъекта на обработку персональных данных
  • Необходимость исполнения договора, стороной которого является субъект
  • Выполнение требований законодательства Российской Федерации
  • Необходимость для защиты прав и законных интересов оператора или третьих лиц

8. Порядок сбора, хранения и передачи данных

Персональные данные пользователя никогда не будут переданы третьим лицам, за исключением случаев, связанных с исполнением законодательства.

  • Данные передаются третьим лицам только с согласия субъекта или для исполнения договора
  • Срок обработки персональных данных определяется достижением целей, для которых были собраны данные
  • По достижении целей обработки персональные данные уничтожаются

9. Трансграничная передача данных

При осуществлении трансграничной передачи персональных данных оператор обязан до начала такой передачи убедиться в том, что иностранным государством обеспечивается надёжная защита прав субъектов персональных данных.

10. Конфиденциальность

Оператор и иные лица, получившие доступ к персональным данным, обязаны не раскрывать третьим лицам и не распространять персональные данные без согласия субъекта, если иное не предусмотрено федеральным законом.
