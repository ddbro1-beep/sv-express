### 3. Обновлён JavaScript
**Файл:** `/packages/landing/assets/js/main.js`

**Добавлены функции:**

1. **Функция перевода (t)**
   ```javascript
   function t(key) {
       return translations[currentLang]?.[key] || key;
   }
   ```

2. **Применение переводов (applyTranslations)**
   - Обрабатывает элементы с `data-i18n`
   - Обрабатывает placeholders с `data-i18n-placeholder`
   - Обновляет атрибут `lang` на `<html>`
   - Обновляет UI переключателя языков

3. **Переключение языка (switchLanguage)**
   - Сохраняет выбор в localStorage
   - Автоматически применяет переводы
   - Обновляет калькулятор и формы

4. **Интеграция с калькулятором**
   - Переведены все сообщения об ошибках
   - Динамические тексты (вес, тарифы)
   - Кнопки и подсказки

5. **Интеграция с формой заявки**
   - Переведены ошибки валидации
   - Сообщения об успехе/ошибке
   - Состояние загрузки

## Как Использовать

### Базовое использование

1. **HTML элементы:**
   ```html
   <!-- Обычный текст -->
   <h1 data-i18n="hero.title.main">Доставка из Европы</h1>

   <!-- Placeholder -->
   <input
       type="text"
       placeholder="Например: SV-2024-00142"
       data-i18n-placeholder="tracking.placeholder"
   >

   <!-- Button -->
   <button data-i18n="leadform.submit">Отправить заявку</button>
   ```

2. **Переключатель языков:**
   ```html
   <span class="lang-switcher" data-lang="ru">RU</span>
   <span class="lang-switcher" data-lang="en">EN</span>
   ```

3. **JavaScript:**
   ```javascript
   // Получить перевод
   const text = t('hero.title.main');

   // Переключить язык
   switchLanguage('en');
   ```

### Добавление новых переводов

1. **Добавьте ключ в translations.js:**
   ```javascript
   const translations = {
       ru: {
           'new.key': 'Новый текст'
       },
       en: {
           'new.key': 'New text'
       }
   };
   ```

2. **Добавьте data-i18n атрибут в HTML:**
   ```html
   <div data-i18n="new.key">Новый текст</div>
   ```

3. **Перезагрузите страницу** - переводы применятся автоматически!

## Сохранение Выбора Языка

Система автоматически сохраняет выбранный язык в `localStorage`:

```javascript
// Ключ: 'sv-express-lang'
// Значения: 'ru' | 'en'

// Получить сохранённый язык
const savedLang = localStorage.getItem('sv-express-lang');

// Установить язык
localStorage.setItem('sv-express-lang', 'en');
```

## Известные Ограничения

1. **Option элементы:** Из-за ограничений HTML, тексты внутри `<option>` не переводятся динамически. Решение - использовать кастомные select'ы.

2. **SEO:** Для полноценного SEO нужны отдельные страницы для каждого языка (index.html, index-en.html).

3. **Контент из API:** Динамический контент (отзывы, FAQ из базы данных) требует дополнительной настройки.
